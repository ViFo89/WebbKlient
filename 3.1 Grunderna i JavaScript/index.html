Grunderna i JavaScript

Externt länkad JavaScript-fil och dess fördelar

Man kan både placera sin JavaScript-kod internt i html-filen eller externt i en egen js-fil. För att skriva intern JS-kod skriver man taggen <script> följt av JS-kod och sedan en sluttag. Intern JS kan vara en fördel om man bara vill använda den koden på endast en sida.

Fördelarna med extern JS-kod är att man kan använda samma kod i flera olika html-dokument. En extern fil är även lättare att läsa och att underhålla. Man lägger till js-filer i sitt html-dokument med hjälp av script-taggen, såhär: <script src="namn.js"></script>

Satser

Man kan säga att en sats i JS är ett steg under körning av koden. Exempel på en sats är alert('Hello World!'); Oftast är en sats på en rad och slutar alltid med ett semikolon. Man kan placera flera satser på en och samma rad men det gör koden svårare att tyda.

Datatyper

I JS finns det tre grunddatatyper. Dessa är nummer, strängar och booleans. Strängar är tecken, ord och meningar. En sträng ska omges av citat-tecken eller apostrof-tecken. Nummer är heltal och flyttal, dessa kan användas för att räkna. Boolean kan endast vara två värden, sant eller falskt.

Exempel på Strängar

  • Kalle
  • Hej på er
  • Jag är 50 år
  • &¤#$
  • a

Exempel på Nummer

  • 23
  • -71
  • 10.2
  • 1.1532

Booleans värden

  • true
  • false

Variabler och arrayer

Variabler är ett utrymme som man namnger och som man sedan kan använda, ändra eller ta bort. För att skapa en variabel anger man nyckelordet var följt av ett namn på variabeln. Såhär: var variabelNamn.

Regler för namngivning av variabler

  • Namnet måste börja med en bokstav, $ eller _.
  • Namnet kan endast innehålla bokstäver, nummer, $ eller _.
  • Variabelnamn är skiftlägeskänslig. VarNamn är inte samma som varNamn.
  • Undvik att använda nyckelord som redan finns i JavaScript.

För att tilldela en variabel ett värde skriver man namnet på variabeln, ett lika med tecken följt av värdet man vill tilldela: var age = 43;

Arrayer

Om man skulle vilja hålla reda på ett lite större antal variabler kan man använda sig av en array. För att skapa en array använder man hakparateser:
var dagar = ['mån', 'tis', 'ons', 'tor', 'fre', 'lör', 'sön'];
Här har vi alltså en array med alla veckans dagar. För att sedan få ut den dagen man vill ha skriver man variabelnamnet följt av hakparanteser och det nummret på i arrayen man vill ha. Så om vi vill ha ut onsdag använder vi nummer 2:
dagar[2];

För att lägga till ett objekt i slutet av en array kan man använda push-kommandot:
var numbers = [1, 2, 3];
numbers.push(4);
Om man vill lägga till ett objekt i början av arrayen använder man kommandot unshift på samma sätt som push.

Operatorer och stränghantering

Operatorer är tecken man använder för att räkna matte i JS. Dessa är +, -, * och /. Dessa fungerar precis som man skulle kunna förvänta sig att de fungerar. Man kan även använda dom tillsammans med variabler:
var tal1 = 2, tal2 = 3;
var tal3 = tal1 * tal2;
Efter detta har tal3 värdet 6. Man kan även använda paranteser om man vill att exempelvis plus ska räknas innan gånger. Vad som räknas först fungerar precis som i matten.

Man kan även använda plus på strängar, detta kallas konkatenering. Det som händer är att man helt enkelt slår ihop de två strängarna till en sammansatt sträng.
var fnamn = 'kalle';
var enamn = 'svensson';
var namn = fnamn + ' ' + enamn;
Resultatet av detta blir att namn tilldelas värdet kalle svensson.

Om man använder plus mellan en sträng och ett nummer är det två saker som händer. För det förstsa görs nummret om till en sträng och för det andra slås de ihop. Om man vill att en string ska behandlas som ett nummer kan man sätta ett plus innan man anger variabelnamnet (utan mellanslag):
var num1 = '2';
var num2 = 4;
var tot = +num1 + num2;
tot får i det här fallet värdet 6. Pluset innan num1 gör att strängen behandlas som ett nummer. Ett annat sätt att göra samma sak är att använda kommandot Number():
var num1 = Number('3');

Flödeshantering (if-satser och loopar)

För att ett program ska kunna göra olika saker när det körs använder man en if-sats. En if-sats används för att kunna göra något som endast ska göras om ett villkor uppfylls. Om man exempelvis skulle vilja skriva ut 'hej' på skärmen endast om man är den 1000e besökaren skriver man såhär:
if(visitors == 1000){
alert('Hej!');
}

Om man vill göra något om inte if-satsen skulle gälla använder man else. Man skriver det då efter if-satsens slutparantes och alla gånger if-satsens villkor är falskt kommer man gå in i else. Om man vill ange flera villkor använder man else if. Så här kan det se ut när man använder alla delar tillsamans:
if(nummer < 5){ alert('Mindre än 5'); }
else if(nummer > 5){ alert('Större än 5'); }
else { alert('Lika med 5'); }

Om man vill upprepa något använder man sig av loopar. Det finns tre olika sorters loopar: while-loop, for-loop och do-while-loop. Man använder while-loopen när man inte vet hur många gånger man behöver upprepa något. Den kommer upprepa koden så länge styrvillkoret är sant. Det kan se ut såhär:
while(x < 10) {
document.write(x);
x += 1;
}
 For-loopen används när man har ett bestämt antal gånger loopen ska upprepas, t.ex. om man ska bläddra igenom en array. Det kan se ut såhär:
for(var num=1; num < enArray.length; num++){
document.write(enArray[num]);
}
 Den sista loopen, do-while fungerar nästan precis som while-loopen men den går igenom koden minst en gång. Den kollar villkoret efter den har gått igenom sin kod. Ser ut såhär:
do{
KOD HÄR!!
}while(x < y)

Funktioner

En funktion är en bit kod som kan anropas och användas flera gånger. Man skriver den på ett separat ställe i koden och anropar den via funktionsnamnet istället. En funktion skrivs såhär:
function funktionsNamn(){
KOD HÄR
}
 Man kan även skicka in variabler in i funktionen som sedan kan användas. Ett exempel är om man har funktionen add som tar emot två stycken tal som sedan ska adderas och efter det ska summan skickas tillbaka. Då ser koden ut såhär:
function add(tal1, tal2){ return tal1 + tal2; }
Med return skickar man tillbaka värdet dit där funktionen anropades ifrån.

Reguljära uttryck

Reguljära uttryck är ett sätt att ange hur en sträng ska se ut. Man kan alltså med hjälp av regexp kontrollera att stängar som telefonnummer, e-postadresser osv. stämmer överrens med ett visst mönster. Man gör detta med hjälp av tecken. Exempelvis använder man \S för att ange vad som helst förutom ett mellanrum, \d för att ange ett nummer. Om man vill göra ett regex-uttryck för ett 10-siffrigt telefonnummer skulle det se ut såhär: \d{3}-\d{7}. Detta gäller för alla telefonnummer som är 3 siffror följt av ett bindestreck och sedan 7 siffror.
