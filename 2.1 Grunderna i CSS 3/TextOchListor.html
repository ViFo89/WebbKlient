Fontfamilj

Serif

Serif fonter är bäst för långa texter. Det tros att serifs små fötter leder ögat från bokstav till bokstav, vilket gör att texten blir mer lättläst. Exempel på Serif-fonts är: Times, Times New Roman och Georgia.

Sans-serif

Sand-serif fonter används oftast för rubriker tack vare deras rena utseende. Exempel är: Arial, Helvetica och Verdana.

Monospace

Monospace används ofta för att visa dator-kod. Varje bokstav i monospace fonter är samma bredd. Exempel är: Courier New, Courier och Monaco.

Enheter

För att ange storleken på en font använder man font-size. Man kan antingen ange storleken i en av de olika storleksenheterna som finns i css eller ange nyckelord. Nyckelorden är xx-small, x-small, small, medium, large, x-large och xx-large. Nyckelordet medium är webläsarens grundstorlek för fonten. Färg

Man kan ange färgen på texten med attributet color. För att ange vilken färg man vill använda finns ett par olika sätt att göra det på. Ett sätt är hexadecimalt, då anger man en färgs värde med en hashtag följt av det hexadecimala talet som representerar färgen. Exempel. svart = #000000, vit = #ffffff, röd = #ff0000. Det hexadecimala talet representerar hur mycket röd, grön och blå man vill ha i färgen. Man kan även ange samma värde via decimaltal, det gör man såhär: grön = rgb(0,255,0). Man kan aven använda procent: blå = rgb(0,0,100%). Vill man ha genomskinlig text anänder man rgba, där a är alfa(genomskinligheten): rgba(50%, 25%, 100%, .5). Alfa anges med ett tal mellan 0-1.

Teckensnitt

  • font
  • font-family - För att ange en font på en sida använder man attributet font-family: följt av den font man vill använda. Man brukar ofta välja att ange flera olika fonter ifall den man egentligen vill använda inte finns på sidan. Det görs såhär: font-family: Arial, Helvetica, sans-serif;
  • font-stretch
  • font-size - Med detta attribut anger man vilken storlek man vill ha på fonten.
  • font-size-adjust
  • font-style - Här anger man om man vill att fonten ska renderas i bold eller italic. Värden man kan ange är: bold, italic och oblique.
  • font-variant - Specificerar vilket fontface fonten man valt har. Värden man kan välja är: normal, small-caps och inherit.
  • font-weight
