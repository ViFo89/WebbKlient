Marginaler, Kantlinjer och Padding

Varje html-element innehåller egenskaper som beskriver hur din webbläsare ska rendera dem. webbläsaren ser alla html-element som små lådor och i lådorna finns texten, bilderna eller vad det nu är för sorts tag. I bilden ovanför så skulle själva innehållet vara content. Padding är utrymmet mellan t.ex. bilden och "ramen". Border är linjen som ritas runt paddingen och man kan välja att rita eller inte rita bordern på en sida. När man anger background-color fylls content, border och padding. Slutligen är margin som är det utrymme som separerar ett element från ett annat.

Marginaler

  • margin - Här anger man marginalen för ett element. Man kan ange marginalen på dessa sätt:
    • margin: 25px 50px 75px 100px;
      • topp-marginalen blir 25px
      • höger-marginalen blir 50px
      • botten-marginalen blir 75px
      • vänster-marginalen blir 100px
    • margin: 25px 50px 75px;
      • topp-marginalen blir 25px
      • höger- och vänster-marginalen blir 50px
      • botten-marginalen blir 75px
    • margin: 25px 50px;
      • topp- och botten-marginalen blir 25px
      • höger- och vänster-marginalen blir 50px
    • margin: 25px;
      • Alla sidor blir 25px
  • margin-top - Anger specifikt toppen-marginalen.
  • margin-right - Anger specifikt höger-marginalen.
  • margin-bottom - Anger specifikt botten-marginalen.
  • margin-left - Anger specifikt vänster-marginalen.

Paddning

  • padding - Denna fungerar likadant som margin fast med padding istället. Alltså läggs det till utrymme mellan ramen och innehållet. Man kan även skriva padding på samma sätt som margin, alltså: padding:4px 3px 2px 0; och det fungerar på samma sidor som beskrivet i margin.
  • padding-top - Används för att lägga till padding på överdelen av ett element.
  • padding-right - Används för att lägga till padding på högersidan av ett element.
  • padding-bottom - Används för att lägga till padding på undersidan av ett element.
  • padding-left - Används för att lägga till padding på vänstersidan av ett element.

Outlines

  • outline - Fungerar som border men tar ingen plats. Med det menas att ingen höjd eller brädd läggs till på elementet. En annan skillnad är att man inte kan välja vilka sidor som ska ha outline, alla sidor får det. Här är två exempel som har exakt samma kod forutom att den första div'en har border och den andra har outline.
    padding:5px;margin5px;border:5px solid black;
    padding:5px;margin5px;outline:5px solid black;

    Det som händer här är att outline-boxen renderas som 10px kortare än border-boxen eftersom 5px från höger och vänster sida inte räknas med.
  • outline-color - Används för att ange färgen på outline.
  • outline-style - Används för att ange stilen på outline. Här kan de samma stilar som border användas.
  • outline-width - Används för att ange tjockleken på outline.

Synlighet

  • visibility - Används för att dölja element på en hemsida. Värdena som kan ges är hidden och visible. Det finns även collapse som endast används på tabeller (fungerar bara i IE och Firefox). När man döljer ett element med visivility: hidden; ligger fortfarande elementet kvar på sidan, det blir alltså bara en tom del där elementet egentligen skulle vara.
  • clip - Används för att skapa en rektangel för att visa bara en del av ett element. Koden ser ut såhär: clip: rect(5px, 100px, 40px, 10px);. Det första måttet är topp offset, det andra måttet är bredden på rektangeln, tredje måttet är höjden och det sista måttet är vänster offset. Alltså kommer elementet med den här koden bara visas från 5px från toppen, 10px från vänster och allt om är 100px till höger på bilden kommer att döljas och allt som kommer 40px ner kommer att döljas. Fungerar endast på absoluta plaserade element.
  • display - Används för att ändra hur element ska visas. Man kan tillexempel ändra block-level element (som paragrafer) så dom blir inline istället eller så kan man göra tvärtom och ändra så inline element(som länkar) visas som block-level element. Exempel:

    Paragraf Ett

    Paragraf Två

    Paragraf Tre

    Länk EttLänk TvåLänk Tre
    Här ser vi att paragraferna (som är block-element) visas som inline-element och länkar som är inline-element visas som block-element.
  • overflow - Används för att specificera vad som ska hända med text som går över sin content-area. Värdena är visible, hidden, scroll och auto.

    Visible gör att texten fortsätter utanför sin content-area.

    Hidden gör att texten försvinner utanför sin content-area.

    Scroll gör att man får en scroll.

    Auto gör att man får en scroll, men bara om det verkligen krävs för att visa resten av innehållet.

Multippla bakgrundsbilder

För att få multippla bakgrundsbilder lägger man helt enkelt bara till ett komma mellan varje bild när man laddar in dem. Såhär: background-image: url(image1.jpg), url(image2.jpg);. För att sedan ändra deras egenskaper sätter man bara komman emellan värden: background-repeat: no-repeat, repeat-y;. Nu har image1.jpg satts till "no repeat" och image2.jpg har satts till "repeat-y".

Här är koden för exemplet ovanför: background: url(img1.png) left repeat-y, url(img2.png) top right repeat-y;

Skuggor

Skuggor används för att få ett element att framstå som om det ligger lite ovanför bakgrunden. Man gör en skugga genom att använda box-shadow: 4px 6px 8px rgba(0,0,0,.75). Första värdet är horisontella offset, andra värdet är vertikala offset, tredje värdet är skuggradien och det sista värdet är skuggfärgen. Man kan ange negativa värden för att få skuggan att ligga åt andra hållet. Vill man att skuggan ska gå innåt i lådan skriver man inset innan det första värdet. Man kan även lägga till ett fjärde värde (läggs in innan färgen) som anger spridningen. 10px i spridning gör att skuggan ögas med 10px utåt.

box-shadow: 4px 6px 8px rgba(0,0,0,.75);
box-shadow: -4px -6px 5px #004500;
box-shadow: inset 0px 0px 8px 2px black;
box-shadow: 0px 0px 10px 10px blue;

Gradienter

Gradienter är när man har en färg och sedan går över i en annan:

background-image: linear-gradient(left,black,white);

För att det ska fungera på alla webbläsare krävs det att man anger ett prefix:

  • background-image: -webkit-linear-gradient(left,black,white);
  • background-image: -moz-linear-gradient(left,black,white);
  • background-image: -o-linear-gradient(left,black,white);
  • background-image: -ms-linear-gradient(left,black,white);
Man kan även ange ett gradertal som anger riktningen av färgskiftningen:
background-image: linear-gradient(90deg,#FFaaaa,#AA0000);
Man kan även använda fler än två färger.
background-image: linear-gradient(right,red,green,yellow,orange);
Man kan ange vart skiftningarna ska ske. Här anger man att efter 10% ska det skiftas till gul, sedan ska nästa skiftning ske vid 90%.
background-image: linear-gradient(left, #900, #FC0 10%, #FC0 90%, #900);
För att få färgskiftning som upprepar sig använder man repeating-linear-gradient.
background-image: repeating-linear-gradient(bottom left, #900 20px, #900 30px, #900 40px);
