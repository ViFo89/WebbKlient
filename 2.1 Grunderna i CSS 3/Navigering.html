2.5 Navigering

Länkttillstånd är att en länk (<a>-taggen) på en hemsida kan befinna sig i olika tillstånd. Dessa tillstånd är: En länks vanliga tillstånd (a:link), som en länk är när man inte har markören på länken och att man inte klickat på länken. Ett andra tillstånd är när man har markören på länken (a:hover). Tredje tillståndet är den korta stunden när man klickat på länken (a:active) och det sista tillståndet är en besökt länk (a:visited). För varje tillstånd kan man ange en egen stil. Här har vi ett exempel:

En Länk

Här ser man att länken är först orange, det är när man inte har besökt länken innan. När man för markören över länken blir bakgrunden grön, klickar man på den blir texten svart och fet medans bakgrunden blir gul och om den är besökt är textfärgen röd. Koden i css filen ser ut såhär: 

a:link#liEx
{
	color: orange;
}
a:visited#liEx
{
	color: red;
}
a:hover#liEx
{
	background-color: green;
}
a:active#liEx
{
	color: black;
	font-weight: bold;
	background-color: yellow;
}

Man kan ändra hur markören ska se ut när man för musen över ett element. Exempel: En länk. Detta göra man med attributet cursor.

Hur man skapar en knappliknande länk

En knapp

Man kan själv välja hur knappen ska se ut, men de huvudsakliga attributen är border och background-color. Bara genom att ange bakgrundsfärg, textfärg och border har vi skapat knappen ovan. För att ge en effekt av att man klickar på knappen kan man sedan ändra färg på den. Här är koden för knappen: 

a.button
{
	background: #B1B1B1;
	color: #FFF;
	font-weight: bold;
	border: 4px solid;
	border-color:#DFDFDF #666 #333 #858585;
}
a.button:hover
{
	background: #C5C5C5;
}
a.button:active
{
	background: #DFDFDF;
	border-color:#333 #666 #DFDFDF #858585;
}

Hur man skapar en länk med grafik

En länk

För att skapa en vacker länk med hjälp av grafik använder man länkens background-image attribut. Man får tänka på att ange no-repeat och kontrollera bakgrundsbildens placering med background-position. För att placera text i länken använder man padding. Här är koden för knappen här ovanför: 

a#linkG
{
	border:1px dashed gray;
	background: #E7E7E7 url(../icon1.png) no-repeat left center;
	background-size: 35px;
	font-weight: bold;
	padding: 10px 7px 10px 45px;
	text-decoration: none;
	color: black;
}
a#linkG:visited
{
	color: gray;
}
a#linkG:hover
{
	background-color: #C7C7C7
}
a#linkG:active
{
	background-color: #B7B7B7
}

Navigationslistor

En navigationslista är i grunden vanliga länkar. Länkarna är ofta indelade som listföremål (>li<) i en oordnad lista. Eftersom man inte vill styla alla oordnade listor, list-items och länkar på sidan är det en bra idé att peka ut dem med en klass eller ett id. Detta är enklast genom att ge ul-taggen klassen eller id't. Man brukar sedan ta bort punkterna från listan med list-style-type:none; och nollställa vänster margin och padding. Man gör nollställningen eftersom webläsare indenterar list-items, endel indenterar med padding och endel gör det med margin. Här är den kod vi skrivit hittills: 

ul.nav {
	list-style-type: none;
	padding-left: 0;
	margin-left: 0;
}

Vertikal navigationslista

Eftersom <a> är ett inline element är bredden på länken så stor som sitt innehåll. Man kan heller inte påverka paddingen och margin på ett inline element. Därför bör man göra om länkarna till block element med display: block;. Genom att göra det till ett block element blir det samma bredd på alla knappar och man gör också hela området klickbart.
